Ce jeu est mon adaptation du jeu Candy Crush réalisée en java. Le but est d'aligner 3, 4 ou 5 images identiques en cliquant sur deux images voisines pour les intervertir. Les images ne seront interverties que si l'une d'elle s'alignera avec au moins 2 images identiques après interversion. Après une minutes de jeu, le score s'affiche et la partie est terminée.

Les points imortants de cette version du jeu sont les fenêtres, les grilles et les threads.

Amélioration intéressante : utiliser un tableau double pour la grille. Ici, j'ai calculé les positions avec des divisions et des modulos, ce qui rend le code plus long et compliqué. Cela n'enlève rien aux compétences de développement que je souhaite démontrer à travers ce code.
